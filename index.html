Introduction

Multi-core era is here to stay and many programmers will be developing parallel software. But still not everyone is trained in parallel programming. In this course, we will study the state-of-art multicore processor architectures, learn how to program parallel processors systems, and learn how to put all these vast machine resources to the best use. With the understanding of the core concept of architecture, and significant hands-on experience with common parallel programming model, we are able to develop real applications on real hardware that can achieve performance and scalability, which is of importance to joining the work force or graduate school.
